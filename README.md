# P-CAB 현황 및 임상적 시사점 | Evidence Landscape

P-CAB(Potassium-Competitive Acid Blocker) 관련 최신 임상논문, 임상시험, 진료지침을 정리한 정적 웹페이지입니다.

## 배포 방법 (GitHub Pages)

1. 이 저장소의 파일들을 GitHub 저장소에 업로드합니다 (`index.html` 파일명 유지).
2. 저장소 `Settings` → `Pages` → `Branch`를 `main` / `root`로 설정 후 저장합니다.
3. 1~2분 후 `https://[사용자명].github.io/[저장소명]/` 주소로 접속 가능합니다.

## 구성

- `index.html` — 단일 정적 HTML 파일 (내부 CSS/JS 포함, 별도 빌드 과정 불필요)
- 외부 리소스: Google Fonts(IBM Plex Sans KR, Noto Serif KR) — 인터넷 연결 필요
